# Valkyrie 语言参考 (Language Reference)

Valkyrie 是一门现代的函数式编程语言，融合了代数效应系统、强类型系统以及极致自由的语法设计。本参考文档详细介绍了语言的各个组成部分。

## 目录结构

### [1. 核心语法 (Core Syntax)](./syntax/index.md)
基础语法构造、变量定义、控制流以及核心解析逻辑。

### [2. 类型系统 (Type System)](./type-system/index.md)
代数数据类型（ADT）、泛型、关联类型、依赖类型以及统一的类型求值引擎。

### [3. 编程范式 (Paradigms)]
- [面向对象 (Object-Oriented)](./object-oriented/index.md)：类、继承、Trait 系统、Widget 组件。
- [面向函数 (Function-Oriented)](./function-oriented/index.md)：匿名函数、模式匹配。
- [响应式编程 (Reactive)](./reactive-programming/index.md)：信号（Signal）、流（Stream）、异步。

### [4. 效应系统 (Effect System)](./effect-system/index.md)
代数效应、异步、错误处理、IoC。

### [5. 元编程 (Meta-Programming)](./meta-programming/index.md)
宏系统、编译时计算、单元系统。

### [6. 生命周期与内存 (Lifetime)](./lifetime/index.md)
所有权、生命周期、分配器。

### [7. 模块系统 (Module System)](./module-system/index.md)
项目管理、工作区、外部函数接口。

### [8. 编程模式 (Patterns)](./patterns/index.md)
构建器模式等高级编程范式。
