# 函数式编程 (Functional Programming)

Valkyrie 是一门深受函数式编程思想影响的语言，提供了丰富的函数式特性，使代码更加简洁、可组合且易于测试。

## 核心概念

### 一等公民函数

在 Valkyrie 中，函数是一等公民。这意味着函数可以：
- 赋值给变量
- 作为参数传递给其他函数
- 作为其他函数的返回值

### 不变性

Valkyrie 鼓励使用不可变数据。虽然支持可变性（通过 `mut` 关键字），但默认情况下变量是不可变的，这有助于减少副作用并提高并发安全性。

### 纯函数

虽然 Valkyrie 允许副作用，但推荐编写纯函数。纯函数的输出仅取决于输入，且不产生可观测的副作用。

## 主要特性

### [匿名函数与闭包](./anonymous-functions.md)

匿名函数是没有名称的函数，而闭包是可以捕获其定义环境变量的匿名函数。Valkyrie 提供了简洁的闭包语法和自动参数推断。

### [模式匹配](./pattern-match.md)

强大的模式匹配系统允许你根据数据的结构进行分支处理，支持解构元组、数组、对象以及自定义联合类型。

### 高阶函数

高阶函数是接受函数作为参数或返回函数的函数。常见的内置高阶函数包括 `map`, `filter`, `fold` 等。

```valkyrie
let numbers = [1, 2, 3, 4, 5]

# 使用高阶函数进行组合
let result = numbers
    .filter { %x % 2 == 0 }
    .map { %x * %x }
    .fold(0) { %acc + %item }
```

### 函数组合

通过将简单函数组合成复杂函数，可以构建高度模块化的系统。

## 优势

- **简洁性**: 减少模板代码，使逻辑更清晰。
- **可组合性**: 通过组合小型、单一职责的函数来构建复杂功能。
- **可测试性**: 纯函数易于进行单元测试，因为它们不依赖外部状态。
- **安全性**: 减少状态管理带来的复杂性和潜在错误。
