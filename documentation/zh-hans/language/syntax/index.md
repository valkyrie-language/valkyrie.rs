# 核心语法 (Core Syntax)

本章节介绍了 Valkyrie 语言的基础语法构造，包括变量定义、控制流、字面量以及特殊的符号处理逻辑。

## 章节内容

- **[定义 (Definitions)](./definitions.md)**：如何声明命名空间、变量、函数、类和联合类型。
- **[控制流 (Control Flow)](./control-flow.md)**：条件分支（if/else）、循环以及模式匹配基础。
- **[字面量 (Literals)](./literals.md)**：数值、字符串、布尔值等基础数据的表示方法。
- **[大括号 (Braces)](./braces.md)**：大括号在命名空间、控制流和字符串插值中的用法。
- **[原生 UI 语法 (V-Grammar)](./v-grammar.md)**：利用 ApplyBlock 实现的声明式 UI 构建模式。
- **[XML 语法 (X-Grammar)](./x-grammar.md)**：UI 结构的视觉投影语法。
- **[模板语法 (T-Grammar)](./t-grammar.md)**：用于代码生成和文本处理的模板系统。
