# Valkyrie Language Reference

Valkyrie is a modern functional programming language that integrates an algebraic effect system, a strong type system, and an extremely free syntax design. This reference documentation provides a detailed introduction to the various components of the language.

## Table of Contents

### [1. Core Syntax](./syntax/index.md)
Basic syntax constructs, variable definitions, control flow, and core parsing logic.

### [2. Type System](./type-system/index.md)
Algebraic Data Types (ADT), generics, associated types, dependent types, and a unified type evaluation engine.

### [3. Patterns](./patterns/index.md)
Advanced programming paradigms such as the Builder Pattern.

### [4. Paradigms](./index.md)
- **[Object-Oriented](./object-oriented/index.md)**: Classes, inheritance, Trait system, Widget components.
- **[Function-Oriented](./function-oriented/index.md)**: Anonymous functions, pattern matching.
- **[Reactive Programming](./reactive-programming/index.md)**: Signals, Streams, asynchronous operations.
- **[Meta-Programming](./meta-programming/index.md)**: Macro system, compile-time computation, unit system.

### [5. Advanced Features](./index.md)
- **[Effect System](./effect-system/index.md)**: Algebraic effects, asynchronous operations, error handling, IoC.
- **[Lifetime & Memory](./lifetime/index.md)**: Ownership, lifetimes, allocators.
- **[Module System](./module-system/index.md)**: Project management, workspaces, foreign function interface.
