# Valkyrie Language Documentation Project

This project is the official documentation repository for the **Valkyrie Programming Language**, built with [VitePress](https://vitepress.dev/).

## Project Introduction

Valkyrie is a modern functional programming language focused on providing a powerful **Algebraic Effects** system and **strong type safety**. This project contains all official learning resources, reference guides, and example code for the Valkyrie language.

## Documentation Structure

- **[Getting Started](./guide/)**: Quick start with Valkyrie, environment configuration, and basic syntax.
- **[Language Features](./language/)**: Deep dive into core features like algebraic effects, pattern matching, metaprogramming, and module systems.
- **[Real-world Examples](./examples/)**: Various application scenario examples ranging from Web development to deep learning.
- **[FAQ](./faq.md)**: A compilation of common questions and answers encountered during development.

## Development and Preview

This project is powered by VitePress, supporting live preview and static site generation.

### Install Dependencies

In the `projects/valkyrie-document` directory, install using pnpm:

```bash
pnpm install
```

### Start Development Server

```bash
pnpm dev
```

Once started, you can access `http://localhost:5173` in your browser for live preview.

### Build Static Site

```bash
pnpm build
```

The build artifacts will be saved in the `.vitepress/dist` directory.
