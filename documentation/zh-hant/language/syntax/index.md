# 核心語法 (Core Syntax)

本章節介紹了 Valkyrie 語言的基礎語法構造，包括變數定義、控制流、字面量以及特殊的符號處理邏輯。

## 章節內容

- **[定義 (Definitions)](./definitions.md)**：如何宣告命名空間、變數、函式、類別和聯合型別。
- **[控制流 (Control Flow)](./control-flow.md)**：條件分支 (if/else)、迴圈以及模式比對基礎。
- **[字面量 (Literals)](./literals.md)**：數值、字串、布林值等基礎資料的表示方法。
- **[大括號 (Braces)](./braces.md)**：大括號在命名空間、控制流和字串插值中的用法。
- **[原生 UI 語法 (V-Grammar)](./v-grammar.md)**：利用 ApplyBlock 實作的宣告式 UI 建構模式。
- **[XML 語法 (X-Grammar)](./x-grammar.md)**：UI 結構的視覺投影語法。
- **[範本語法 (T-Grammar)](./t-grammar.md)**：用於程式碼生成和文字處理的範本系統。
